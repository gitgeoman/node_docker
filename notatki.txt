-tworzenie kontenera:                   docker run -p 3000:3000 -d -v PATHTOFOLDER:PATHINCONTAINER--name node-container node-app-image
-tworzenie kontenera z vlumemnem
-tworzneie kontenera z zabezpieczeniem danych (zakaz modyfikacji folderu node modules)
                                        docker run -v D:\Programowanie\Docker\yt2\:/app -v /app/node_modules -p 3000:3000 -d --name node-container node-app-image 
-tworzenie kontenera z read only bindmount (dodaje :ro po -v path:ro)
                                        docker run -v D:\Programowanie\Docker\yt2\:/app:ro -v /app/node_modules -p 3000:3000 -d --name node-container node-app-image 

-przekazywanie zmiennych w trakcie tworzenia (zmienna PORT): docker run -v D:\Programowanie\Docker\yt2\:/app:ro -v /app/node_modules --env PORT=4000 -p 3000:4000 -d --name node-container node-app-image 
        - uwaga można też plikiem:       docker run -v D:\Programowanie\Docker\yt2\:/app:ro -v /app/node_modules --env-file ./.env -p 3000:4000 -d --name node-container node-app-image
